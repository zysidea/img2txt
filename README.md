## 转换图片为文本，输出ASCII到控制台
Usage:
```sh
  imgtotxt -p[path/url] -w[width] -r[highlight the content]
```
The default vaue of `-p` is google' logo.

The default vaue of `-w` is 80.

The default vaue of `-r` is false.
